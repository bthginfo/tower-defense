# Game Design Document

## Vision

Critter Covenant ist ein freundliches, schnelles Tower Defense über improvisierte Teams magischer Tierhüter. Zufällige Beschwörung erzeugt Spannung; Covenants belohnen flexible Kompositionen statt nur identischer Einheiten.

## Vertical-Slice-Loop

Essence ausgeben → Hüter beschwören → Pfad verteidigen → drei identische Hüter mergen → Covenants aktivieren → alle fünf Wellen einen Boss besiegen. Niederlage tritt bei null Herzen ein.

## Content

Sechs Hüter decken Nature, Fire, Water, Storm, Guardian und Alchemist ab. Dornenbund (2× Nature) sowie Flammenwirbel (Fire + Storm) erhöhen aktuell den Angriffsschaden; spätere Iterationen erhalten eigene Flächen-/Ketteneffekte.
